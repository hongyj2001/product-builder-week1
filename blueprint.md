
# Lotto Number Generator

## Overview

A simple, visually appealing web application to generate random lottery numbers. The application will feature a clean, modern design and provide an intuitive user experience.

## Project Outline

### Style and Design

*   **Layout:** A centered, clean layout that is mobile-responsive.
*   **Color Palette:** A vibrant and energetic color palette will be used.
*   **Typography:** Expressive typography will be used to emphasize key elements.
*   **Visual Effects:** Subtle shadows and gradients will be used to create depth and a premium feel.
*   **Interactivity:** The "Generate" button will have a "glow" effect on interaction.

### Features

*   **Number Generation:** Generate 6 unique random numbers between 1 and 45.
*   **Display:** The generated numbers will be displayed in styled circles.
*   **User Interaction:** A single button to trigger the number generation.
